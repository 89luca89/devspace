```bash
#######################################################
############ devspace remove image ####################
#######################################################
Removes one or all images from a devspace:
devspace remove image default
devspace remove image --all
#######################################################

Usage:
  devspace remove image [flags]

Flags:
      --all    Remove all images
  -h, --help   help for image
```
