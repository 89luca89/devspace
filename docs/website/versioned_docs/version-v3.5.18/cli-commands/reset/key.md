```bash
#######################################################
############### devspace reset key ####################
#######################################################
Resets a key for a given cluster. Useful if the key
cannot be obtained anymore. Needs cluster access scope

Examples:
devspace reset key my-cluster
#######################################################

Usage:
  devspace reset key [flags]

Flags:
  -h, --help              help for key
      --provider string   The cloud provider to use
```
